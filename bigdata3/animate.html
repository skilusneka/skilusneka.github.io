动效

					动效适用场合：
						1.需要提示内容变化
						2.对页面上较为单调的元素进行修饰
					动效的持续时间：
						通常是0.5s
						最长1s，最短0.3s
					使用动效的方法：
						1.使用css的transition样式
							一次性的动画优先使用该方法
							给元素添加transition样式,当元素样式改变时自动产生补间动画；
							例：
								div{
									transition：0.5s;
								}	
						2.使用css的关键帧动画
							适用于持续性的动画，如头部移入效果
							例：
								@keyframes run{
									from{width:0%}to{width:100%}
								}
								div{
									animation: run;
									animation-duration:1s;
									animation-play-state:infinite;
								}
						3.使用js
							配合计时器改变元素的样式，用以上两种方式难以达到效果时使用该方法。
							注意清除定时器！部分浏览器有不清除定时器动画越走越快的bug。
							例：
								var t =setInterval(function(){
									if(t>100){
										$(this).css('width',t+'px');
										t++;
									}else{
										clearInterval(t);
									}
									
								},500)；
						4.使用插件
							使用插件绑定的动画方法，视具体情况使用。
